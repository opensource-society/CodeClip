# 🤝 Contributing to CodeClip

Welcome! 👋  
We're excited you're here and interested in contributing to **CodeClip**, a lightweight and useful Chrome extension for managing code snippets.

This project is in its early stage — so **you’re welcome to help build it from the ground up!**

---

## 🚀 Getting Started

1. **Fork the repo**
2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUROWNUSERNAME/CodeClip
   cd CodeClip
   ```
3. **Create a new branch**
   ```bash
   git checkout -b my-feature
   ```  
4. **Start coding!** Edit files, add features, fix bugs.
5. **Commit and push**
   ```bash
   git commit -m "Add: my feature"
   git push origin my-feature
   ```
6. **Open a Pull Request** on the main repository

---

## 🛠 What Can You Work On?

Since the project is new, feel free to contribute:

* 🔧 Setup improvements (folder structure, manifest config)
* 🧱 UI (popup, vault, themes)
* 💾 Storage (save/retrieve snippets)
* 🎨 Icons and design
* 🐞 Bug fixes
* 📄 Documentation

We’ll also add `good first issues` and `help wanted` tags as the project evolves.

---

## 📚 Guidelines

* Keep code clean and readable
* Use meaningful commit messages
* Comment your code when needed
* Respect the [Code of Conduct](CODE_OF_CONDUCT.md)

---

## 🙌 Need Help?

If you're stuck or unsure, feel free to open an issue or ask in a discussion. We're here to help!

Let’s build something useful and open together. 💡
